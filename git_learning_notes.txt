get the version of git :
git --version   

paramétrage de l'utilisateur :
git config --global user.name "Maxime Song"
git config --global user.email "songmaxime@gmail.com

initialisation de git dans le projet :
git init

connaître l'état du projet git :
git status

faire une photo du projet :
git add (file_name)
git commit -m "Reason of the commit"

obtenir les commit du projet :
git log

création d'une branche :
git branch (branch_name)

lister les branches :
git branch

changer de branches :
git checkout

rapatrier une branche :
retourner à la branche en question
git merge (branch_name)
git branch -d (to remove the old branch)
